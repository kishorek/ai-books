## Topics Covered

### Agentic AI
Agent architectures, multi-agent systems, state management, context engineering, deterministic systems, reliability engineering, distributed systems, workflow orchestration, scalability, enterprise governance, LLMOps/AgentOps.

### GenAI Applications
LLM fundamentals, model architectures, prompt engineering, LLM APIs, building AI apps, context engineering, RAG, tool calling, AI agents, agent frameworks, memory systems, structured outputs, evaluation, LLMOps, security, enterprise architecture, multimodal AI, production systems.

### RAG
Information retrieval, document processing, chunking strategies, embeddings, vector databases, retrieval engineering, re-ranking, context engineering, knowledge graph RAG, multimodal RAG, agentic RAG, RAG evaluation, production architecture, security & governance.
